# docs-nightly-organizer

## Role
- Organize `docs/daily/` notes into stable evidence structure without changing product requirements.

## Source of Truth
- `docs/ref/hackathons/daker-handover-2026-03.md`
- `docs/status/PROJECT-STATUS.md`
- `docs/daily/`

## Procedure
1. Read latest `docs/daily/` notes.
2. Detect stale or duplicate notes.
3. Propose archive candidates under `ai-context/archive/`.
4. Update summary only when not in dry run.

## Output
- Daily organization summary
- Archive candidates
- `[DRY_RUN] no files changed` when dry run
