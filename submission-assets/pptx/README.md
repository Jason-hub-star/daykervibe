# Expedition Hub PPTX Generator

이 폴더는 제출용 PPTX 초안을 자동 생성하는 스크립트를 담고 있다.

## Usage

```powershell
npm install
npm run build
```

생성 결과:

- `out/expedition-hub-submission-deck.pptx`

## Source Inputs

- `submission-assets/pdf/outline.md`
- `submission-assets/pdf/slide-copy.md`
- `docs/submission/asset-audit.md`
- `output/playwright/*.png`
