# Expedition Hub Video Scene Plan

기준일: 2026-04-01 (KST)

이 문서는 제출용 유튜브 롱폼 영상에서 어떤 화면을 어떤 의도로 쓰는지, 그리고 자막을 어디에 배치하는지 고정하는 운영 문서다.

## 공통 레이아웃

- 상단 좌측: 장면 제목과 kicker
- 상단 우측: `Scene 01/11` 같은 진행 표시
- 우측 메인: 실제 서비스 화면 still
- 좌측 메인: 핵심 설명, 심사 포인트, evidence
- 하단 중앙: 내레이션 자막

## 자막 세이프존

- 위치: 하단 중앙
- 여백: 화면 맨 아래에서 약 `132px` 위
- 폭: 최대 `1160px`
- 줄 수: 기본 2줄
- 스타일: 반투명 짙은 박스 + 큰 한글 고딕
- 원칙: 서비스 UI를 직접 가리지 않도록 still 위에 덮되, 하단 네비/버튼과 겹치는 장면은 크롭으로 피한다

## 편집 원칙

- still을 단순 정지 이미지로 두지 않고, 약한 `zoom-in` 또는 `pan`으로 리듬을 준다.
- War Room과 상세 페이지 장면은 가장 오래 노출한다.
- 자막은 설명을 반복하지 않고, 내레이션의 핵심만 1~2줄로 압축한다.
- 강조 텍스트는 상단/좌측 카드에서 처리하고, 하단 자막은 듣는 내용을 따라가기 쉽게 유지한다.
